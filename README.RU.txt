WadDirectory/WarStudio
Цель проекта -- создание кроссплатформенного редактора wad-файлов (файлов данных в играх и портах, основанных на Doom), который позволит открывать и редактировать лампы (lumps) напрямую во внешних программах. Я считаю, что редактор wad-файлов (или ламп-менеджер) должен быть реализован именно таким образом, т.к. встроенные редакторы никогда не достигнут уровня уже существующих самостоятельных текстовых процессоров, редакторов скриптов, изображений и т д.

В рамках проекта будет разработано две программы: WadDirectory и WarStudio.
WadDirectory будет консольной программой для создания и редактирования wad-файлов, до некоторой степени схожей с DeuTEX. Ее задачами станут конвертация между общепринятыми сейчас форматами и форматами, использованными в Doom, а также, конечно, архивирование лампов в wad-файлы и их извлечение.
В будущем WadDirectory станет основой для WarStudio.
WarStudio будет полноценным графическим ламп-менеджером для Doom. Интерфейс менеджера планируется реализовать с использованием библиотеки Qt.

---

Технические подробности

Код проекта пишется на подмножестве C++11, доступном в MS Visual Studio Express 2012.
На данный момент проект имеет следующие зависимости:
- Boost 1.51
- GraphicsMagick 1.3.17 Q16
- Filesystem из TR2 (в VS входит в состав стандартного SDK)

Все библиотеки скомпилированы и подключаются как multi-threaded dll
